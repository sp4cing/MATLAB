This study has been prepared for those who want to quickly work the classification with Deep Learning on MATLAB. 
There are some images of 5 different classes under the <data_set> folder. 
These images were downloaded from the <https://tiny-imagenet.herokuapp.com/> site under the name <Tiny ImageNet>.
The study consists of Custom Network Design and Fine Tuning projects.

It is important that this study is an introduction study for DL. 
Python should be preferred for detailed and advanced DL projects.

If the subfunctions (basic_conv,findLayersToReplace,ect.) don't work, 
please click on the utils folder with the "mouse 2", and then select "Add to path -> Select Folder and Subfolders".
